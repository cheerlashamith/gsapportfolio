# Shamith Cheerla's Portfolio

Welcome to my personal portfolio repository! This project showcases my journey, projects, achievements, and experiences as a Software Developer. It's built with modern web technologies, focusing on immersive storytelling, high-performance animations, and responsive design.

## 🚀 Live Demo

You can view the live portfolio here: **[https://shamith-portfolio.netlify.app/](https://shamith-portfolio.netlify.app/)**

## ✨ Features

- **Immersive Storytelling:** Beautiful, butter-smooth scroll animations powered by GSAP and Framer Motion.
- **Modern UI/UX:** Glassmorphism design elements, responsive dock menus, and a clean, dynamic layout.
- **Projects Showcase:** Highlights of my best work including StockEase, MindMitra, Windows 11 Portfolio, and more.
- **Hackathons & Achievements:** Infinite-scrolling timeline of hackathons and a dedicated section for academic and technical milestones.
- **Dynamic Contact Form:** Animated and interactive contact form with custom success notifications.

## 🛠️ Tech Stack

- **Framework:** React with Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** GSAP (GreenSock Animation Platform) & Framer Motion
- **Icons:** FontAwesome & Custom SVGs

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/cheerlashamith/portfolio-v2.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd portfolio-v2
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`
